【Java】安装IntelliJ IDEA

发表于2021-12-19|更新于2022-09-05
|字数总计:1.5k|阅读时长:5分钟|阅读量:

没有安装JDK配置Java环境的请看我上一篇文章。

Java 初学者为了能更好的掌握 Java 代码的编写，我们一般会选用一款高级记事本类的工具作为开发工具， 而实际项目开发时，更多的还是选用集成 IDE 做为开发工具，比如当下最流行的两款工具：Eclipse IDE、IntelliJ IDEA。前者是免费的开放源码的项目，后者是收费的商业软件（专业版免费试用30天，社区版免费使用但功能少），本篇笔记安装的是后者的专业版。

软件下载

  • Jetbrains官网：https://www.jetbrains.com/
  • IDEA下载页：https://www.jetbrains.com/idea/
  • 其他版本：https://www.jetbrains.com/idea/download/other.html

下载安装分为exe和zip，我选择的Ulitmate版本的exe程序，当然你也可以选择下载zip安装包，本篇笔记以exe可执行程序为例。

版本需选择与自己找到的破解方法相适配的对应版本，激活方式网上有很多。

学生认证激活

如果是在校学生，并且有EDU邮箱，可以使用学生身份免费试用，续期一年，时间到可继续申请。

  • 可以去 Github 申请GitHub student benefits学生权益包。有的学校邮箱域名被拉黑了，原因你懂的，详见：JetBrains的域名黑名单

  • 申请通过后前往GitHub Student Developer Pack，点击相关内容跳转到JetBrains的Free Educational Licenses页面，点击【Apply now】按钮申请即可，申请方式选择Github。

2022.1.3安装激活

图文教程1：https://tech.souyunku.com/?p=30970

图文教程3：https://tech.souyunku.com/?p=49896&type=IDEA

  • 先把老版本卸载干净，卸载程序是安装目录下/bin文件夹中的Uninstall.exe，两个选项都勾选上

  • 卸载完成后，看一下安装目录还在不在，将其彻底删除

软件安装

  • 到官网下载合适版本，比如2022.1.3 - Windows (exe)

  • 运行安装包，选择适当的安装目录

  • 勾选“创建桌面快捷方式”

  • 点击Install安装

  • 勾选运行IDEA，点击Finish

  • 等待IDEA启动

  • 若弹出下方弹框，点击“不导入配置”

  • IDEA 运行成功后，会弹出下面的对话框，提示我们需要先登录 JetBrains 账户才能使用。这里我们先不管，先点击 Exit 退出，准备开始引入激活补丁。

破解激活

  • 使用激活补丁前，需要清空 IDEA 以前使用过的激活方式（非常重要）

  • 下载解压激活补丁文件夹，打开文件夹如下：

  • 复制整个激活补丁文件夹到硬盘某个位置，可以放到D盘或E盘，打开\scripts目录，文件如下：

  • 其中install开头的是安装脚本，uninstall开头的是卸载对应脚本

    1. install.sh：Mac/Linux 系统点击运行脚本
    2. install-all-users.vbs：为系统所有用户安装，不太推荐
    3. install-current-user.vbs：Windows 系统点击运行脚本，为当前用户安装破解补丁

  • 我是Windows，所以我双击执行install-current-user.vbs。

  • 稍等片刻弹出Done弹窗表示补丁安装成功，点击确定。

  • 该破解补丁文件夹，不要移动位置，也不要删除，也千万不要升级 IDEA 2022（非常重要）

  • 重启 IDEA 2022, 选择Activation code填入IDEA激活码，完成激活，Close关闭即可。
