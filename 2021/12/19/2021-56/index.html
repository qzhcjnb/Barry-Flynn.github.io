Java开发工具

Java 初学者为了能更好的掌握 Java 代码的编写，我们一般会选用一款高级记事本类的工具作为开发工具， 而实际项目开发时，更多的还是选用集成 IDE 做为开发工具，比如当下最流行的两款工具：Eclipse IDE、IntelliJ IDEA。前者是免费的开放源码的项目，后者是收费的商业软件（专业版免费试用30天，社区版免费使用但功能少），本篇笔记安装的是后者的专业版。

1. 浏览官网

Jetbrains官网：https://www.jetbrains.com/
IDEA下载页：https://www.jetbrains.com/idea/
其他版本：https://www.jetbrains.com/idea/download/other.html

2. 版本下载

下载安装分为exe和zip，我选择的Ulitmate版本的exe程序，当然你也可以选择下载zip安装包，本篇笔记以exe可执行程序为例。

【点击下方直接下载exe可执行程序】：

版本2020.2.2 - Windows (exe)
安装包大小：663MB

【点击下方直接下载zip安装包】：

版本2020.2.2 - Windows ZIP Archive (zip)
压缩包大小：768MB
解压后大小：约1.5G
