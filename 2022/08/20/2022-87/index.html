【GitHub】关于我申请学生开发者包时发生的事儿

发表于2022-08-20|更新于2022-08-23
|字数总计:3.5k|阅读时长:11分钟|阅读量:

Join GitHub Global Campus

  • 加入 GitHub 全球校园！
    加入 GitHub 全球校园，为从事科技行业做好准备。Global Campus 将通过让您访问行业工具、活动、学习资源和不断壮大的学生社区，帮助您获得所需的实用行业知识。

GitHub student benefits

GitHub学生权益：

  1. 【免费】学生时期 GitHub Pro
  2. 宝贵的GitHub Student Developer Pack合作伙伴优惠
  3. 为合格的申请者提供 GitHub Campus Expert 培训

(1) GitHub Pro

GitHub Pro，即 GitHub专业版：

除了 GitHub Free (GitHub免费版) 为个人帐户提供的功能外，GitHub Pro (GitHub专业版) 还包括：

  • 通过电子邮件提供 GitHub 支持（GitHub Support via email）
  • 3,000 分钟 GitHub Actions（3,000 GitHub Actions minutes）
  • 2GB GitHub 包存储（2GB GitHub Packages storage）
  • 私有存储库中的高级工具和见解：（Advanced tools and insights in private repositories:）
    • 所需的拉取请求审阅者（Required pull request reviewers）
    • 多个拉取请求审阅者（Multiple pull request reviewers）
    • 受保护的分支（Protected branches）
    • 代码所有者（Code owners）
    • 自动链接参考（Auto-linked references）
    • GitHub 页面（GitHub Pages）
    • 维基（Wikis）
    • 存储库洞察图：脉冲、贡献者、流量、提交、代码频率、网络和分叉（Repository insights graphs: Pulse, contributors, traffic, commits, code frequency, network, and forks）

(2) GitHub Student Developer Pack

GitHub学生开发者包：

  • 学习像专业人士一样发布软件。亲身体验是无可替代的。但对于大多数学生来说，现实世界的工具可能成本高昂。这就是我们与一些合作伙伴和朋友一起创建 GitHub Student Developer Pack 的原因。

(3) GitHub Campus Expert

GitHub校园专家：

  • 丰富你的大学技术社区，通过 GitHub 的培训、指导和支持，学习在校园内建立和发展多样化技术社区的技能。

Get your GitHub student benefits

GitHub Pro申请资格：

  • 目前就读于授予学位或文凭的学习课程，例如高中、中学、学院、大学、家庭学校或类似的教育机构。
  • 拥有一个可验证的学校颁发的电子邮件地址或上传证明您当前学生身份的文件。
  • 拥有 GitHub 用户帐户。
  • 至少 13 岁。

提供可用性：

  • 所有合作伙伴优惠均受合作伙伴公司的条款和条件约束，并且可能并非在所有地区都可用。


Your ID should include a date that verifies your current enrollment. Make sure the image is clear and easy to read; if it looks blurry, please take a new photo and upload it again.


If you don’t have a student ID, or it doesn’t include a date, you can upload a letter on school letterhead or any documentation with a date that demonstrates your current enrollment.

无教育邮箱解决方法

  • 填写的GitHub Pro申请表单中需要使用教育邮箱，否则无法下一步。

  • 然而蛋疼的是很多学校并没有给本科生发放教育邮箱，懂得都懂……

  • 如果某学校强制必须输入教育邮箱才能下一步，可以把刚刚选中填入的学校名称删除后边中文部分，只保留英文名称。此时【Continue】按钮就可以正常点击了：

  • 在地图下方的搜索框中输入大学名称，填写相关信息，点击【Continue】按钮

  • 上传的图片需要包含日期信息，证明目前是在校学生。可以点击【Upload an image】上传学生证或联名银行卡照片，有日期的那一面，最好也把带头像的那一面P到一张图上，图片中也最好需要包含学校名称。选择证明类型为 Dated school ID，点击【Process my application】按钮。


  • 点击完后稍等几秒，进行提交前的预检验。

  • 然后点击【Continue anyway】按钮，完成申请。

  • 片刻后会收到一封邮件

有教育邮箱了欧耶！

2022年8月21日晚20时48分，这是一个不寻常的一刻。

下午16时左右，才使用学信网信息给GitHub提交申请，正等待人工审核结果，竟然晚上就收到通知，河师大给全体在校生开通EDU教育邮箱了！喜大普奔！喜大普奔！！！

河师大本科生EDU邮箱开通公告

全校本科生：
  为给广大学子提供更好的网络信息服务，提升我校信息化建设水平，方便于学生申请境外访学、学习等各方面的交流与沟通，现我校为全体本科生开通EDU邮箱，邮箱统一格式为：学号@stu.htu.edu.cn。

现将有关事项通知如下：

  • 开通流程：无需申请，在教务系统中已完善手机号的同学自动开通。
  • 激活方式：（具体操作请参考附件中的操作指南）
    1. 在教务系统中查询自己的手机号是否完善或者存在错误。
    2. 在浏览器中打开以下链接：http://mail.stu.htu.edu.cn/
    3. 首次需选择手机号登录，登录成功则可扫码绑定微信，之后便可使用微信进行登录。
  • 邮箱客户端：EDU邮箱可通过企业微信进行访问。移动端或PC端搜索企业微信应用并下载，使用绑定的微信进行授权登录。
  • 注意事项：
    1. 在使用过程中要遵守国家相关法律法规，并妥善保管好个人账号及密码。对于来历不明的邮件，务必谨慎打开。
    2. 教务系统中没有完善手机号的同学请及时进行完善，否则会出现手机号未绑定用户的情况。（该步骤操作均需等待48小时后，再进入邮件系统激活个人EDU邮箱）
    3. 尚未登录过邮箱系统，且需要变更手机号的同学，请登录教务系统中进行手机号的变更（该步骤操作均需等待48小时后，再进入邮件系统激活个人EDU邮箱）；登录过邮箱系统，且绑定过微信的同学，请直接在企业微信上修改手机号码即可。
    4. 毕业后，该邮箱仍可免费使用，但将于毕业一年之后开始清除，条件是超过两年以上未登录。一旦清除邮箱内容不可恢复、也不可重新开户。

在使用过程中出现其他异常，请发送“学院、年级、专业、学号、姓名、手机号、性别、问题描述”至邮箱mail@htu.edu.cn，联系人：柴老师

信息化建设与管理办公室

2022年8月18日

EDU邮箱知识:

  • 高校师生身份标志之一：@stu.htu.edu.cn是我校为在校全体师生开设的带有大学后缀的教育专属电子邮箱。

  • 申请高校或投稿必备：高校师生在申请国外高校、学术投稿、学术会议注册和学术网站登录时，多建议使用带有学校表示的@XXX.edu.cn域名邮箱。如很多国外知名期刊在接收投稿时，由服务器首先进行邮箱过滤，个人邮箱的投稿多被认为是垃圾邮件进行处理，所以@htu.edu.cn域名邮箱是必备条件。

  • 享受教育专属资源优惠:使用@XXX.edu.cn域名邮箱可以获得一定期限内的专享优惠，例如由GitHub、DigitalOcean、Namecheap、Amazon等企业联合提供的GitHub服务包，包括Pycharm、IntelliJ IDEA、ReSharper等各类开发工具和集成工具在内的JetBrains学生包，Autodesk、Adobe、Microsoft、OriginLab等提供的免费应用资源。百度云、腾讯云、阿里云提供的教育优惠，以及苹果、微软等品牌众多软硬件产品折扣等等。

  • 提升权威性和归属感：高校在招生招聘、日常宣传等工作中使用xxx@XXX.edu.cn域名邮箱，能够提升学校的官方形象和宣传力度，可信性更强。据了解，很多高校为以往毕业生的校友保留了原有邮箱的使用权或申请校友邮箱的权利，以此作为终身与母校联系的纽带，一定程度上增加了大家的归属感。

用教育邮箱重新申请！

果不其然，昨天的申请没有通过。没有教育邮箱看来还是很难，即便使用了学信网的在线验证报告。不知道会不会是由于中文页面的问题…

既然学校发教育邮箱了，那就用Edu邮箱重新提交一次吧。

没有Edu邮箱的朋友，听说可以去某宝、某多多等平台买一个

  • 在申请页面点击添加电子邮件地址，跳转到GitHub个人账号邮箱设置页面。添加Edu邮箱地址，通过邮件通过验证。

  • 返回申请页面，邮箱选择教育邮箱。

  • 下面的学校名称已通过Edu邮箱自动填写好，只需填写 How do you plan to use GitHub? 内容，然后 Continue。

  • 接下来没有了通过地图选位置的步骤，直接跳到了上传拍摄图片那一步。但好像只让【Take a picture】，没有【Upload an image】按钮。

  • 笔记本的摄像头很模糊，尽可能拍了清晰的一张学生证。提交时报错，是因为开了VPN导致与学校所在地不符，如果远程学习需要提供线上证明。

  • 关闭VPN重新申请，上传图片【Upload an image】按钮又出现了……上传了学生证照片，提交时报错，因为以前使用过照片。请上传一张新照片。……

  • 重新刷新，换了一张学生证照片。提交时报错，因为

    1. 您的文件似乎没有包含表明当前学术隶属关系的日期。对于使用非标准日历的国家/地区，您可能需要在带有转换日期的文档旁边捕获原始文档。您可以在图像中包含多个文档，只要它们清晰易读。
    2. 如果您已完成GitHub 用户个人资料，您更有可能获得验证。
    3. 您的图片应包含 2018 至 2026 范围内的日期

  • 用QQ截图手动P了一下学校名称、入学日期和毕业日期的英文翻译，再次提交，提交成功。

  • 接下来等回复吧，接着等吧……

  • 不到5分钟，我刷新申请页面发现服务器报错502，以为是网络问题，结果一看主页：

我敲，用Edu邮箱竟然这么神速！这跟前两次的差别不是一点半点啊！
