宝塔面板申请Let's Encrypt泛域名证书(使用阿里云DNS)

发表于2022-04-05|更新于2022-04-11
|字数总计:312|阅读时长:1分钟|阅读量:

新建了几个站点，发现宝塔SSL的三张免费申请额度已经用完了，

好吧，那就申请 Let’s Encrypt 的 SSL 证书吧。

打开站点 设置 - SSL - Let's Encrypt

选择验证方式：DNS验证(支持通配符)

选择DNS接口：阿里云DNS（因为我域名是在阿里云买的）

弹出窗口：设置【阿里云DNS】接口

前往阿里云控制台，鼠标悬浮右上角头像处，点击 AccessKey管理

云账号 AccessKey 是您访问阿里云 API 的密钥，具有账户的完全权限，请您务必妥善保管！不要以任何方式公开 AccessKey 到外部渠道（例如 Github），避免被他人利用造成 安全威胁。强烈建议您遵循 阿里云安全最佳实践，使用 RAM 用户（而不是云账号）的 AccessKey 进行 API 调用。

选择 继续使用AccessKey，点击 创建AccessKey，进行手机或人脸验证。

成功后就可以看到查看 Secret 的 AccessKey ID 和 AccessKey Secret，复制到宝塔面板的弹出窗口即可。

勾选：自动组合泛域名

域名：勾选站点域名

点击申请，稍等片刻，部署成功，强制HTTPS。
